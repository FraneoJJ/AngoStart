Projeto: AngoStart

Missão:
Criar a principal plataforma angolana de validação e crescimento de negócios baseada em localização e inteligência artificial.

Arquitetura:
- Backend: API REST
- Banco: MySQL
- Frontend: React + Vite
- Google Maps API
- WebSockets
- Stripe
- Resend
- Alibaba Cloud ID Verification
- Documentero

Funcionalidades principais:

1. Validação de ideia por IA
   - Questionário dinâmico
   - Análise de viabilidade
   - Pontos fortes e fracos
   - Sugestões de melhoria

2. Integração com Google Maps
   - Localização do usuário
   - Análise por região

3. Acompanhamento contínuo do negócio
   - Checklist automático
   - Estratégias de crescimento

4. Orientação legal em Angola
   - Constituição de empresa
   - Estrutura societária
   - Procedimentos na AGT

5. Sistema de assinatura

6. Chat, voz e video chamadas

Regras:
- Arquitetura modular
- API REST bem documentada
- Separação clara entre camadas
- Segurança obrigatória
- Implementar por módulos
